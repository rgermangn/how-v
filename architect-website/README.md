# Architect Website

This project is a personal website for an architect, showcasing their portfolio, services, and contact information. The website is structured to provide a user-friendly experience while highlighting the architect's work and expertise.

## Project Structure

```
architect-website
├── assets
│   ├── css
│   │   └── styles.css
│   ├── js
│   │   └── scripts.js
│   └── fonts
├── pages
│   ├── about.html
│   ├── contact.html
│   ├── portfolio.html
│   ├── services.html
│   └── index.html
└── README.md
```

## Features

- **Responsive Design**: The website is designed to be fully responsive, ensuring a seamless experience on both desktop and mobile devices.
- **Portfolio Showcase**: A dedicated portfolio page that displays completed projects with filtering options and detailed views.
- **Service Listings**: A services page outlining the various architectural services offered, along with a clear work process and FAQ section.
- **Contact Form**: An easy-to-use contact form on the contact page, along with direct contact options and an embedded Google Maps location.

## Setup Instructions

1. Clone the repository to your local machine.
2. Open the `index.html` file in your web browser to view the homepage.
3. Navigate through the pages using the provided links in the navigation menu.

## Additional Notes

- Ensure that all assets (CSS, JS, fonts) are correctly linked in the HTML files.
- Customize the content in each HTML file to reflect the architect's personal brand and projects.
- Test the website across different browsers and devices for compatibility.
